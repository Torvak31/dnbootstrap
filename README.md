# Disclaimer

All credit goes to the original creator of this project (https://github.com/VSMobile/dnbootstrap). I added to this with AI so I could create a better controlling system for my liking.

Not all features of the original are present here (because I am not experienced enough to integrate my modifications in later versions). Use the original one if you want an editable layout, fewer bugs and more.

What I (AI) added:
- non configurable layout with joystick
- mod importing by selecting "open with" on a zip file in a file manager (not all mods work)
- minecraft mobile touch controls (short touch = right mouse click, long touch holding = left mouse holding)

For SHIFT and CTRL to work you need to assign the buttons inside the controls settings.
